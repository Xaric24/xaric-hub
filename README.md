# 🎮 Xaric Hub

Universal Roblox exploit hub with auto-game-detection and one-click script launching.

| Script | Game | Features |
|--------|------|----------|
| 🌷 **Bloom v3** | The Garden Frontier | Auto-Farm, Grid Plant, Profit Tracker, ESP |
| 🌱 **Reseeder** | Be A Seed? | Auto-Collect, Auto-Run, Gamepass Spoof, Speed Hack |
| 🐟 **Cobalt v3** | BE A FISH BAIT! | Auto-Fish, Auto-Train, QTE, Movement |
| ⚔ **Warhead v1** | Missiles vs Cities | Auto-Fire, Auto-Farm, Movement, Teleport |
| 🔧 **Cobalt GUI** | Any Game | Script Executor, Console, Explorer, Remote Spy |

## Usage

```lua
loadstring(readfile("xaric_hub.lua"))()
```

1. Execute `xaric_hub.lua` in your executor
2. Hub auto-detects your current game
3. Wait 5s for auto-launch or click **LAUNCH** on any card
4. Hub closes and loads the selected exploit

## Hub Features

- **Auto-Detection** — Matches game PlaceName to the right script
- **5s Auto-Launch** — Detected game launches automatically (click another to cancel)
- **Dark Theme UI** — Animated cards, hover effects, smooth transitions
- **Manual Override** — Launch any script regardless of detected game
